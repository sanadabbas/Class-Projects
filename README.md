# Class Projects

A collection of Python projects created for class assignments.

## Projects

### SnakeGame.py
A classic Snake game implementation in Python. Guide the snake to eat food and grow longer while avoiding collisions with walls and yourself.

### birthday.py
A birthday-related utility program that handles birthday calculations and information.

### purchasedogs.py
A program for managing dog purchases or a dog store system with purchase functionality.

## Getting Started

Each project can be run independently. Make sure you have Python installed on your system.

To run any project:
```bash
python filename.py
```

## Requirements

- Python 3.x
